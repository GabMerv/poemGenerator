# poemGenerator
-> générer un poème à partir d'un extrait des fleurs du mal de Baudelaire

## Fonctionnement

Le programme se réfère au fichier **data.txt**, il est donc possible de changer ce fichier avec des poèmes d'une autre personne pour changer le style d'écriture.
Il est possible d'utiliser les différentes fonctions implémentées pour changer de type de poème (différentes strophes, ...)

La génération de poème se fait à l'aide d'une étude fréquentielle d'appiration des mots en connsaissance du mot précédent.
Aurement dit, il s'agit simplement d'une étude statistique et les expressions crées par le programme ne veulent pas toujours dire grand chose.

Toutefois, nous retrouvons des rimes en fin de vers et une possibilité de configurer quels types de poèmes on recherche.
_Par exemple: des odes_
